# User Stories – Цифровая карта лояльности

1. Как клиент FortePay я хочу видеть текущий баланс баллов в приложении, чтобы понимать сколько бонусов накоплено.  
2. Как клиент я хочу получать push-уведомление о начислении баллов, чтобы чувствовать ценность программы.  
3. Как маркетолог я хочу настраивать кампании с дополнительными бонусами по MCC, чтобы стимулировать покупки в конкретных сегментах.  
4. Как администратор лояльности я хочу видеть историю списаний баллов, чтобы выявлять подозрительные операции.  
5. Как клиент я хочу быстро обменивать баллы на купоны, чтобы использовать их в магазинах-партнёрах.

Разработать систему автоматических накоплений для клиентов банка с умными советами
Клиенты не умеют правильно планировать финансы и копить деньги, нет инструментов для автоматизации накоплений, сложно отслеживать финансовые цели
Веб-платформа и мобильное приложение для создания финансовых целей, автоматических переводов на сберегательный счет, анализа расходов и умных рекомендаций по накоплениям
Клиенты банка, финансовые консультанты банка, отдел маркетинга, отдел разработки продуктов, аналитики по клиентскому опыту
Клиент (создание целей и накопление), финансовый консультант (помощь клиентам), администратор системы (настройка правил накоплений), аналитик (анализ данных)
Разработка приложения для iOS и Android, модуль автоматических переводов, система финансовых целей, аналитика расходов, интеграция с банковскими системами, модуль рекомендаций
Инвестиционные советы, кредитные продукты, интеграция с внешними банками, криптовалютные накопления
100000 активных пользователей в первый год, средняя сумма накоплений 50000 тенге на пользователя, 60% пользователей достигают своих финансовых целей, оценка приложения 4.0/5
Автоматический перевод возможен только с основного счета на сберегательный. Минимальная сумма перевода 1000 тенге. Максимальная сумма накоплений 10 млн тенге на одного пользователя. Проценты на накопления начисляются ежемесячно
Бюджет 3 млн тенге, срок разработки 6 месяцев, интеграция только с банковскими системами данного банка, поддержка только тенге
Низкая мотивация пользователей к накоплениям, риски технических сбоев при автоматических переводах, проблемы с аналитикой расходов из-за приватности данных
Пользователи заинтересованы в накоплениях, банк готов предоставить API для интеграции, пользователи согласны делиться данными о расходах для получения рекомендаций
Время обработки автоматического перевода менее 1 секунды, 99.8% доступность системы, шифрование всех финансовых данных, соответствие требованиям ЦБ РК, поддержка до 1000 одновременных пользователей
Пользователь открывает приложение и создает финансовую цель (например, накопить на отпуск). Устанавливает сумму и срок. Система предлагает план накоплений и автоматические переводы. Ежемесячно происходит автоматический перевод указанной суммы. Система отслеживает прогресс и показывает график. При достижении цели пользователь получает уведомление
